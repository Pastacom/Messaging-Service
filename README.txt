Проект для моделирования системы отправки сообщений.
Особенности моего проекта:
1. При генерации пользователей и сообщений JSON файлы перезаписываются, имейте это в виду.
2. Генерируется всегда по 10 пользователей и сообщений.
3. Позволяется создавать сообщения самому себе. То есть, когда почта отправителя равна 
почте получателя. Однако при генерации почты всегда будут различны.
4. Пустые списки не возвращаются, будут возвращены статусы отличные от 200.
5. Чтобы все точно запускалось необходимо запускать MessagingService, а не IIS Express.
